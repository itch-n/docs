# System Algorithms

## Learning Objectives
- Understand algorithms used in distributed systems
- Implement consistent hashing and load balancing
- Apply system-level algorithmic thinking

## Key Topics
- Consistent Hashing
- Load Balancing Algorithms  
- Rate Limiting Algorithms
- Consensus Algorithms (Raft, Paxos)
- Distributed Hash Tables

## Advanced Concepts
- Vector clocks and logical time
- Merkle trees for data integrity
- Gossip protocols for information dissemination
